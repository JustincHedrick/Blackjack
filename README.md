<h2>Getting Started </h2>

<h3>Instructions </h3>
<details open>
<summary>How to Start a hand</summary>
<ol>
<li>The game message board advises the player to place a bet.</li>
<li>After a player has placed a bet, the deal button appears.</li>
<li>Clicking deal will then display cards with the hand values displayed in the message board.</li>
</ol>
</details>

<details>
<summary>Adding cards to the player and dealer hands</summary>
<ol>
<li>The user will be prompted to 'hit' or 'stay'</li>
<li>By clicking the 'hit' button, a card will be added to the players and an updated value will be placed in the message board</li>
<li>When the player clicks the 'stay' button, the dealer will be dealt cards and the game will be evaluated for a winner</li>
</ol>
</details>

<details>
<summary>Reset the current hand and redeal</summary> 
<li>After the winner has been evaluated, 'new deal' will appear.</li>
<li>When the player clicks 'new deal' the wager will be evaluated and deducted from the bank total.</li>
<li>The player will the be asked to place a bet.</li>
</details>
